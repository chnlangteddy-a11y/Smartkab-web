## 1. 首页 Banner 图片

### Banner 1 - 主视觉
**尺寸：** 1920x1080px (16:9)
**风格：** 工业科技感，深色背景

**指令：**
```
创建一张工业能源存储设备的专业产品展示图：
- 背景：深蓝色渐变（#0D1117 到 #1A202C），带有细微的网格纹理
- 主体：户外储能柜产品，放置在工业场景中（工厂屋顶或太阳能电站）
- 光效：左侧蓝色科技光晕，右侧暖橙色阳光照射
- 氛围：专业、高端、工业级
- 细节：产品表面有金属质感，周围有轻微的光粒子效果
- 文字区域：左侧预留40%空间用于叠加标题文字
```

### Banner 2 - 解决方案场景
**尺寸：** 1920x1080px
**风格：** 应用场景展示

**指令：**
```
创建一张工商业储能应用场景图：
- 场景：现代化工厂屋顶，太阳能板阵列 + 储能柜组合
- 时间：黄昏时分，天空有橙色到蓝色的渐变
- 产品：多个白色储能柜整齐排列
- 光效：温暖的阳光从侧面照射，形成长影
- 氛围：可持续能源、绿色科技
- 文字区域：右侧预留30%空间
```

### Banner 3 - 技术实力
**尺寸：** 1920x1080px
**风格：** 技术细节展示

**指令：**
```
创建一张技术细节展示图：
- 背景：深色工业背景，带有电路板纹理（低透明度）
- 主体：储能柜内部结构透视图（3D渲染风格）
- 元素：电池模组、BMS系统、逆变器组件
- 光效：蓝色科技光线条连接各组件

```

## 设计规范总结

### 配色方案
| 用途 | 颜色代码 | 说明 |
|------|----------|------|
| 主色 | #0066CC | 科技蓝，品牌主色 |
| 辅色 | #00A3FF | 电光蓝，强调色 |
| 点缀 | #FF6B00 | 能量橙，CTA按钮 |
| 深色背景 | #0D1117 | 页脚、深色区域 |
| 浅色背景 | #FAFBFC | 页面背景 |
| 文字主色 | #1A202C | 标题文字 |
| 文字次色 | #6B7785 | 正文文字 |

### 字体规范
- **标题：** Barlow（粗体，600-700）
- **正文：** Inter（常规，400-500）
- **代码/数据：** JetBrains Mono（等宽）

### 间距规范
- 区块间距：py-16 lg:py-24（64px / 96px）
- 容器宽度：max-w-7xl（1280px）
- 卡片间距：gap-8（32px）
- 内边距：p-6 到 p-8（24px-32px）

### 圆角规范
- 按钮：rounded-xl（12px）
- 卡片：rounded-2xl（16px）
- 图片：rounded-xl（12px）
- 标签：rounded-full（完全圆角）
